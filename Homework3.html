Homework #3 CDO (copula)

 

Choose 10 stocks and do the following:

  1. Get CDS quotes and use one-period (back-of-envelope) formula to get (unconditional) PD
  2. Use copula to get conditional PD
  3. Apply Vasicek to compute loss function
  4. Compute expected tranche losses using the Black-Scholes model (see below **)
  5. [bonus] Compute expected tranche losses using the loss function derived in #3 above

 

 

** Use the BS model to calculate expected losses of tranches of a CDX which is $1.25 billion in size.  EL (i.e. mean of the distribution) is $100 million of the entire portfolio.  Simply set S0=100.  Volatility is 0.3, risk-free rate is 5% (i.e. 0.05), time horizon is 1 year (T = 1).  Evaluate the expected losses of equity tranche (0~3%), mezzanine tranches (3~7, 7~10, 10~15), senior tranche (15%~30%).  [Hint: Start with the senior equity tranche which is a call option BS(K5) and work you way back to equity tranche which is BS(K1) – BS(K0).  Mezzanine value is BS(K2) – BS(K1), etc.  Note that K1 = $37.5 mil, K2 = $87.5 mil, K3= $125 mil, K4 = $187.5 mil.]
